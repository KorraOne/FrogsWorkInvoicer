# Invoice App (grandparents)

Personal/family Australian GST invoicing tool. This is the **original app**. No billing, accounts, or commercial features.

- **Entry:** `app.py`
- **AppData:** `%APPDATA%\InvoiceApp\`
- **Build:** `..\build.ps1` from repo root → `dist\InvoiceApp\InvoiceApp.exe`
- **Docs:** [`../docs/app-description.md`](../docs/app-description.md)

The commercial fork lives in [`../client_app/`](../client_app/) and does not import from this folder.
