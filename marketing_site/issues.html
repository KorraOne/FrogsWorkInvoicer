Frequent issues and troubleshooting

Try these before you contact support. Video guides cover many steps in short clips.

Quick checks

  1. Open the Cloud app at app.frogswork.com (or your Windows desktop window) and sign in.
  2. Check you are online for sign-in, sync, and email send.
  3. Confirm your subscription under Settings → Account in the app.
  4. Sending invoices. Use Send on the invoice. Email goes through FrogsWork Cloud.

Install on your phone

iPhone / iPad: Recommended browser: Safari → open app.frogswork.com → Share → Add to Home Screen.

Android: Recommended browser: Chrome (not Samsung Internet) → open app.frogswork.com → menu → Install app or Add to Home screen.

Windows or phone shows a security warning

These screens look scary but are normal for apps that are not on the Microsoft Store or Google Play yet. You can continue if you downloaded FrogsWork from frogswork.com or installed from app.frogswork.com.

Windows: SmartScreen / unknown publisher

Windows SmartScreen may say the app is unrecognized. Choose More info, then Run anyway. The installer is not code-signed yet.

Android: “built for an older version of Android”

This usually happens when installing from Samsung Internet, not from Chrome. Uninstall if needed, open the site in Chrome, and install again. The warning is about how that browser packages the home-screen app, not about FrogsWork itself. See this known Samsung Internet issue.

Windows installer will not run

Download again from Download. Windows may block files from the internet. Right-click the setup file → Properties → Unblock, then run. If you see a security warning instead, see Windows or phone shows a security warning. You need Windows 10 or 11 (64-bit). The Windows app opens the same Cloud UI as the browser.

WebView2 missing

The Windows shell needs Microsoft Edge WebView2. The installer usually installs it. If the app shows a blank window, install the WebView2 runtime and restart FrogsWork.

Cannot reach account server

Sign-in, sync, and email need api.frogswork.com. Check your internet connection and firewall. In the app, open Settings → Account and try again.

Windows shell update failed

Download the latest installer from this site and run it over the existing install, or keep using the browser app at app.frogswork.com.

Sign-in problems

Use forgot password to reset your account password by email.

How do I send a quote?

Quotes are optional. In the Cloud app open Settings → General and turn on Enable quotes. A Quotes tab appears between Home and Invoices. Create a quote or price estimate, email it to the customer, then use Create invoice when they accept.
