Support · Using FrogsWork

Frequent issues

Installer won't run

Download again from Download. Windows may block files from the internet: right-click the setup file, open Properties, tick Unblock, then run. You need Windows 10 or 11 (64-bit).

WebView2 missing

FrogsWork needs Microsoft Edge WebView2. The installer usually installs it. If you see a blank window, install the WebView2 runtime and restart FrogsWork.

Can't reach the billing server

Invoicing within the free tier works offline. Sign-in, sync, and usage above $2,000 ex GST need api.frogswork.com. Check your connection and firewall. Pages should still load quickly. Open Settings → Your account to retry sync.

Update failed

Use Settings → App update → Check for updates when online. If download fails, get the latest installer from this site and run it over your existing install.

PDF missing or wrong folder

PDFs save in the folder you chose at install (usually …\pdfs). Change it in Settings → Data storage. On the send page, use Show PDF in folder.

Can't sign in

Reset your password through KorraOne if you forgot it. If the server is down, you can still use the free tier offline with your local data.
