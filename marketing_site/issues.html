Frequent issues and troubleshooting

Try these before you contact support. Video guides cover many steps in short clips.

Quick checks

  1. Restart FrogsWork. Close it from the taskbar if needed, then open it from the Start menu.
  2. Check you're online if you need sign-in or invoicing after the free trial (20 invoices or $20,000 ex GST).
  3. Check your version in Settings → App update.
  4. Find your data. Settings and invoices are in %APPDATA%\FrogsWork\. PDFs are in your pdfs folder (Settings → Data storage).
  5. Sending invoices. Use Prepare to send, copy the email text, and open the PDF yourself.

Installer will not run

Download again from Download. Windows may block files from the internet. Right-click the setup file → Properties → Unblock, then run. You need Windows 10 or 11 (64-bit).

WebView2 missing

FrogsWork needs Microsoft Edge WebView2. The installer usually installs it. If the app shows a blank window, install the WebView2 runtime and restart FrogsWork.

Cannot reach account server

Invoicing during the free trial works offline. After subscribing, the app caches your subscription and allows up to 14 days offline before requiring verification. Sign-in and subscription checks need api.frogswork.com. Check your internet connection and firewall. Open Settings → Your account and tap Verify subscription.

Update failed

Use Settings → App update → Check for updates when online. If download fails, download the latest installer from this site and run it over the existing install.

Invoice PDF missing or wrong folder

PDFs save under the folder you chose at install (usually …\pdfs). Change it in Settings → Data storage. Use Prepare to send → Show PDF in folder on the send page.

Sign-in problems

There is no self-service password reset yet. Contact support. If the account API is unreachable, you can still use the free trial offline with existing local data.
