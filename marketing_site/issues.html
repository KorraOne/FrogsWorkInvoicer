Frequent issues and troubleshooting

Try these before you contact support. Video guides cover many steps in short clips.

Quick checks

  1. Open the Cloud app at app.frogswork.com and sign in.
  2. Check you are online for sign-in, sync, and email send.
  3. Confirm your subscription under Settings → Account in the app.
  4. Sending invoices. Use Send on the invoice. Email goes through FrogsWork Cloud.

Install on your phone

Open app.frogswork.com in Safari or Chrome. Use the browser menu to Add to Home Screen / Install app.

Windows installer will not run

Download again from Download. Windows may block files from the internet. Right-click the setup file → Properties → Unblock, then run. You need Windows 10 or 11 (64-bit).

WebView2 missing

The Windows shell needs Microsoft Edge WebView2. The installer usually installs it. If the app shows a blank window, install the WebView2 runtime and restart FrogsWork.

Cannot reach account server

Sign-in, sync, and email need api.frogswork.com. Check your internet connection and firewall. In the app, open Settings → Account and try again.

Windows shell update failed

Download the latest installer from this site and run it over the existing install, or keep using the browser app.

Sign-in problems

Use forgot password to reset your account password by email. Promo codes and Stripe subscription trials are entered at checkout, not as an in-app free meter.
