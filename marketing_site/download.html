How to install

  1. Click the download button above and wait for the file to finish downloading.
  2. Open the installer and follow the on-screen prompts. The default location is fine for most people.
  3. Launch FrogsWork from the Start menu when installation is complete.

If Windows shows a security warning about an unknown publisher, choose More info, then Run anyway. The installer is not signed yet.

Your invoices and settings

FrogsWork keeps your data in its own folder on your PC, separate from the program itself. If you uninstall, you can export your invoices to your Downloads folder first.

Automatic updates

When you are online, FrogsWork checks for newer versions and offers to install them from inside the app. Your invoices and settings are not changed by an update.
