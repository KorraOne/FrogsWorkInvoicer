How to install

  1. Click download and wait for the file to finish.
  2. Run the installer. The default location is fine for most people.
  3. Open FrogsWork from the Start menu.

If Windows warns about an unknown publisher, choose More info, then Run anyway. The installer is not signed yet.

Your invoices and settings

Your data lives in AppData on your PC, separate from the program. You can export invoices before uninstalling.

Automatic updates

When online, FrogsWork checks for updates and installs them from inside the app. Your data is not changed.
