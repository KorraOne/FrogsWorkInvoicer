Install steps

  1. Download the zip and extract the FrogsWork folder anywhere (for example Downloads or Desktop).
  2. Run FrogsWork.exe. First launch installs to %LOCALAPPDATA%\Programs\FrogsWork\ and restarts — you can choose a Desktop shortcut then.
  3. Your invoices and settings live in AppData; only the program files move.
  4. If Windows blocks the app after download, right-click the zip → Properties → check Unblock, then extract again.

Requirements

  • Microsoft Edge WebView2 Runtime (usually already installed on Windows 11).
  • Internet optional on the free tier; required for account sign-in and fee-bearing usage.

Updates

When a newer version is available, the app shows a banner after it connects to api.frogswork.com. Choose Update now to download from downloads.frogswork.com and restart. Your AppData is not changed.
