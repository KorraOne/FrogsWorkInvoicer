Install steps

  1. Download and run FrogsWork-—-setup.exe.
  2. Follow the installer. The app installs to %LOCALAPPDATA%\Programs\FrogsWork\ (you can change the folder).
  3. Your invoices and settings live in %APPDATA%\FrogsWork\ — separate from the program files.
  4. If Windows SmartScreen warns about the download, choose More info → Run anyway (the file is not signed yet).

Requirements

  • Microsoft Edge WebView2 Runtime (usually already installed on Windows 11).
  • Internet optional on the free tier; required for account sign-in and fee-bearing usage.

Updates

When a newer version is available, the app shows a banner after it connects to api.frogswork.com. Choose Update now to download from downloads.frogswork.com and restart. Your AppData is not changed.
