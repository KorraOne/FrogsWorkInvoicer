How to install

  1. Click download and wait for the file to finish.
  2. Run the installer. The default location is fine for most people.
  3. Open FrogsWork from the Start menu and sign in with your Cloud account.

If Windows warns about an unknown publisher, choose More info, then Run anyway. The installer is not signed yet. See also security warnings when installing.

Your invoices

Invoice data lives in FrogsWork Cloud (same as the browser app). The shell does not keep a separate Local invoice database.

Automatic updates

When online, the shell can check for updates and install them from inside the app.
